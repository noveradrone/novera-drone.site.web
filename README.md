# Novera Drone - Site vitrine premium

## Stack
- Next.js 14 (App Router)
- TypeScript
- TailwindCSS
- Framer Motion
- GSAP
- Formulaire contact sans API (mailto)

## Installation
```bash
npm install
npm run dev
```

## Variables d'environnement
Non requis pour la version actuelle (envoi via `mailto:`).

## Notes media
- Le hero lit `/public/videos/drone-hero.mp4`.
- Le poster du hero est `/public/images/hero-poster.svg`.
- L'image section A propos est `/public/images/about-drone.svg`.
- Le portfolio utilise `/public/images/portfolio/portfolio-01.svg` a `portfolio-06.svg`.
- Instagram utilise `/public/images/instagram/instagram-01.svg` a `instagram-06.svg`.
- Remplacez ces placeholders par vos vrais fichiers (meme nom recommande).

## SEO local intégré
- Metadata dynamiques
- JSON-LD LocalBusiness
- `sitemap.xml` automatique
- `robots.txt`

## Qualité
- Composants réutilisables
- Responsive mobile-first
